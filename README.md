# ProxUI
ProxUI is a Proxmox tool that installs a basic desktop environment so you can manage and control your VM's and containers all from your server. Primarily intended for those whom want to run Proxmox as a main os on their Desktop/Laptop.

This product is currently still in development, but the code is public in case you decide to install this tool in its beta form (I heavily suggest that you avoid using this tool, espicially on your main Proxmox installation).
